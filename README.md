# Retail Sales Data Analysis & Dashboard Project

## Overview

This is an end-to-end data analytics project where raw retail data was cleaned, transformed, and analyzed to generate business insights. The project demonstrates data wrangling, KPI analysis, and dashboard creation using multiple tools.

---

## Tools & Technologies

* Alteryx (Data Cleaning & Transformation)
* SQL (KPI Analysis)
* Power BI (Dashboard & Visualization)
* Excel (Reporting)

---

## Project Structure

```
data/
  raw_data.csv
  cleaned_data.csv

workflows/
  data_cleaning_workflow.yxmd

sql/
  kpi_queries.sql

dashboard/
  powerbi_dashboard.pbix

screenshots/
  dashboard.png
```

---

## Process Workflow

1. Imported raw dataset and cleaned it using Alteryx workflows
2. Handled missing values, removed duplicates, and standardized data
3. Created new features like age groups and cleaned categorical variables
4. Used SQL to extract key business KPIs
5. Built an interactive Power BI dashboard with filters and scorecards

---

## Key KPIs

* Total Revenue
* Average Order Value (AOV)
* Total Customers
* Revenue by Category
* Seasonal Sales Trends

---

## Dashboard Features

* KPI cards for quick insights
* Seasonal trend analysis
* Category-wise revenue analysis
* Customer distribution by gender
* Interactive slicers (Season, Category, Gender)
* Performance scorecard (Target vs Actual)

---

## Key Insights

* Clothing category generates the highest revenue
* Summer season shows relatively lower sales performance
* Customer distribution varies across gender segments
